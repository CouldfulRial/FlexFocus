# FlexFocus

macOS 可视化专注应用（MVP）。

## 已实现功能

- 中间主区：时钟 + 开始专注按钮。
- 点击开始后先输入任务，支持从历史任务快速选择（纵向完整列表，最近任务在最上方）。
- 开始后正计时；手动结束后记录本次专注。
- 结束专注后询问是否开始休息，休息倒计时为专注时长的 `1/5`，最短 `1` 分钟。
- 专注计时颜色：`#FF5666`，休息计时颜色：`#495F41`（dark 模式自动反色）。
- menu bar 会显示实时计时文本（专注为正计时、休息为倒计时）。
- 右侧专注历史：按日期分组、组内按时间倒序（最新在上），支持右键编辑任务与删除记录。
- 左侧统计栏：
	- 顶部固定 `横坐标区间` 选择：`时 / 天 / 周 / 月`
	- 柱状图（标题：`专注时长`）：
		- `时`：12 个 bar，每个 bar 表示 1 小时
		- `天`：7 个 bar，每个 bar 表示 1 天
		- `周`：7 个 bar，每个 bar 表示 1 周（7 天）
		- `月`：12 个 bar，每个 bar 表示 1 个月
		- 保留横坐标 interval 纵向网格线（不显示横轴文字）
		- 每个 bar 顶部显示环比值（仅数字，不带 `+/-/%`）
		- bar 颜色按环比方向变化：增长 `#2E5339`、负增长 `#30011E`、持平灰色（dark 模式自动反色）
		- 环比对比规则：`时` 对比 24 小时前、`天` 对比 7 天前、`周` 对比 1 周前、`月` 对比上个月
		- 背景高亮规则：`时` 视图高亮 9:00–17:00（工作时间），`天` 视图高亮周一至周五（工作日）
		- Y 轴带时长单位
	- 柱状图下方提供区间翻页：左箭头 / 开始时间 / `-` / 结束时间 / 右箭头（`-` 保持统计栏中间）
		- 翻页步进：`时` ±1 小时、`天` ±1 天、`周` ±1 周、`月` ±1 个月
		- 范围上限约束：结束时间不会超过当前（`时` 不超过当前小时，`天/周/月` 不超过今天）
	- 词云：低饱和度颜色、词频映射字号、随机布局、避免重叠、少量竖排补空隙（dark 模式自动反色）
	- 时间轴（标题：`时间轴`，独立统计，不受横坐标区间影响）：
		- 单条全天时间轴：最左 `00:00`、最右 `23:59`
		- 固定 5 个 tick：`00:00 / 09:00 / 12:00 / 17:00 / 23:59`，并带纵向虚线
		- 仅当所选日期是今天时显示“当前时间”纵向指示线
		- 时间轴下方显示：`共专注 XX 小时，占8小时的Y%`（`XX`、`Y` 使用专注色）
		- 标题右侧可选日期：左/右箭头按天切换；点击日期弹出日历浮窗选择任意日期
- 主窗口最小尺寸已限制：横向不小于三栏最小宽度总和，纵向不小于三块统计图可正常显示的高度。
- 设置窗口（`⌘,`）：
	- `通用` tab：
		- 主题模式：`Light / Dark / 跟随系统`
		- 是否在 `Dark` 模式对主题色执行反色
		- 专注开始时是否开启 DND
		- 专注结束时是否关闭 DND
		- 是否发送休息结束通知
		- 清除所有历史数据
	- `词汇` tab：
		- 词汇模式切换：`黑名单模式（屏蔽词）` / `白名单模式`
		- 当前模式词汇列表（可滚动）+ 搜索过滤
		- `+` 添加词汇、`-` 删除选中词汇
		- `重置默认`：黑名单模式恢复默认屏蔽词；白名单模式清空列表

## 分词规则

- 由 `-` 连接的词（如 `long-term`、`api-v2`）按一个词处理。
- 黑名单模式：命中屏蔽词会被过滤。
- 白名单模式：仅统计白名单中的词汇。

## 运行方式

```bash
swift build
swift run
```

建议在 Xcode 打开本目录后运行，以获得更好的 macOS UI 调试体验。

> 注意：`swift run` 不是 `.app` bundle 运行环境，因此系统通知行为受限。若要完整验证通知与系统集成，建议使用 Xcode 运行或打包成 `.app`。

## 打包成 .app

项目已提供打包脚本：

```bash
chmod +x scripts/package_app.sh
./scripts/package_app.sh
```

输出路径：`dist/FlexFocus.app`

脚本会自动读取项目根目录的 `FlexFocus.png`，生成并注入 `FlexFocus.icns` 作为 App 图标。
若图标转换失败，脚本会回退为无自定义图标打包（不阻塞 `.app` 生成）。

## 系统 Focus 模式接入

由于 macOS 对系统级 Focus 切换的公开 API 有限制，当前实现通过 `shortcuts` 命令触发快捷指令：

- `FlexFocus Start Focus`
- `FlexFocus Stop Focus`

请在系统“快捷指令”中创建上述两个快捷指令，并分别配置为：

- 开启你想使用的专注模式
- 关闭专注模式（或切换回默认）

若快捷指令不存在，App 仍可正常计时和记录，只是不会自动切换系统 Focus。

## 通知说明

- App 启动时（开启通知开关时）会请求系统通知权限。
- 休息倒计时结束后会发送本机通知提醒。
- 已接入基于 CloudKit 的跨设备同步通知：当设备 A 休息结束，会写入 iCloud 事件；设备 B 轮询到新事件后触发本机通知。
- 跨设备通知生效前提：
	- 使用 `.app` 运行（不是 `swift run`）
	- Xcode 中为目标开启 iCloud（CloudKit）能力并完成签名
	- 各设备登录同一 Apple ID 的 iCloud
	- 目标设备也安装并运行 FlexFocus（当前方案由目标设备本地发通知）

## 代码结构

- `Sources/FlexFocus/MainContentView.swift`：三栏主界面与流程串联
- `Sources/FlexFocus/ViewModels/FocusViewModel.swift`：专注/休息状态机与计时逻辑
- `Sources/FlexFocus/Services/SessionStore.swift`：本地 JSON 持久化
- `Sources/FlexFocus/Services/FocusModeService.swift`：Shortcuts 桥接
- `Sources/FlexFocus/Views/*`：UI 组件（计时区、历史区、统计区、任务输入）
- `Sources/FlexFocus/Support/*`：分组与统计聚合
