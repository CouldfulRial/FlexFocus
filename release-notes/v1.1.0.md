# FlexFocus 1.1.0 (2027)

发布日期：2026-03-04

## 本次更新

- 时间轴升级为上下双面板：
  - 上半区展示“当天时间轴”，仅显示红色专注段。
  - 下半区新增“本周时间轴”，按 MON–SUN 七格展示，使用纵向虚线分隔。
- 周时间轴新增“选中日期”高亮框，并随日期切换自动定位到对应周。
- 日期选择统一限制为“不可超过今天”：
  - 日期选择器不可选未来日期。
  - 右箭头前进到今天后自动禁用。
- 词云交互增强：
  - 悬停词高亮改为字体放大。
  - 浮窗动画恢复，内容精简为统计信息（XXh，占YY%）。
  - 小窗口下词云字号自适应，减少文字截断。
- 设置页增强：
  - 词汇搜索无匹配时支持“是否创建该词汇”，并加入二次确认。

## 版本信息

- App 版本：1.1.0
- Build：2027
- 打包产物：dist/FlexFocus.app
